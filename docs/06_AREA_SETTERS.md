# Area de Appointment Setters

## Descripcion

El area de appointment setters sera una zona privada independiente para que cada setter pueda consultar su actividad comercial dentro de Con Sabor Flamenco.

## Objetivo del documento

Documentar las funciones previstas para los setters y establecer que cada setter solo debe acceder a la informacion relacionada con su propia actividad.

## Funciones previstas

Cada setter podra:

- Ver su dashboard.
- Ver sus clientes.
- Ver sus leads.
- Ver sus ventas.
- Ver sus comisiones.
- Ver sus codigos promocionales asignados.
- Solicitar el cobro de comisiones.
- Ver pagos recibidos.
- Descargar justificantes o facturas de liquidacion.

## Dashboard

El dashboard debera resumir la actividad comercial del setter: leads asignados, clientes captados, ventas realizadas, comisiones pendientes y comisiones pagadas.

## Codigos promocionales

Cada setter podra tener uno o varios codigos promocionales asignados desde administracion. Estos codigos serviran para aplicar descuentos a clientes y asociar leads al setter propietario.

## Comisiones y cobros

El setter podra consultar sus comisiones y solicitar el cobro de las que esten disponibles segun las reglas comerciales definidas.

## Reglas y decisiones

- El area de setters sera independiente.
- Cada setter solo vera informacion del setter autenticado.
- Los codigos promocionales se crean desde administracion.
- Las comisiones se calcularan preferiblemente sobre el importe realmente cobrado tras aplicar descuento.
